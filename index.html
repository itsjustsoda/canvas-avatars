Spinning Avatars

I just learned that Canvas has an API so after spending some time learning ImageMagick, I made my Canvas avatar point to a random one of these.

These load out of order and one is probably cached so there will probably be an annoying offset. :(

Feel free to use these are your avatar on Canvas. Right now there's a cron job changing cnvs.johnfresco.com/avatar.gif so you can point your avatar to that for a different color every minute. Maybe I'll eventually make the server truely serve a random color on each request.

I'm not aware of any way to use a URL for your avatar picture except through the API so here's some code for that if you want to use it. 

curl "https://utah.instructure.com/api/v1/users/<id>" \
  -X PUT \
  -F 'user[avatar][url]=http://cnvs.johnfresco.com/avatar.gif' \
  -H 'Authorization: Bearer <token>'

You'll need to generate a token here. "self" should work in place of your id but I haven't tested that yet.
